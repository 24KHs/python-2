## pycharm相关设置

### 一、修改头部模板定义

1、设置PyCharm工具的编码格式：
```
File -->> setting -->> File Encodings -->> IDE encoding：utf-8 
```

2、设置模板文件编码类型：
```
File -->> Setting -->> File and Code Templates
```

在模块里顶部加入下面语句
```
#!/usr/bin/env python
# coding=utf-8
```



### 二、设置python标准编码风格（PEP8）
```
file -->> setting -->> inspections -->> PEP 8 coding style violation
```
