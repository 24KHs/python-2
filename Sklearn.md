## scikit-learn

安装
```
$ sudo apt-get install build-essential python-dev python-setuptools python-numpy python-scipy libatlas-dev libatlas3gf-base
$ sudo apt-get install python-matplotlib
$ sudo apt-get install python-sklearn
```

测试
```
$ nosetests -v sklearn
```
